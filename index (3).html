03 — Output

Printing information
with console.log()

console.log() is the developer's way of printing a message to the browser console. Think of it as a window you open to check what your program is doing.

Why use it?

See what's happening inside your code

Programmers use it to check values, test ideas, and find mistakes — it's the first tool you learn and the one you'll use forever.

script.js
// Printing a string (text) console.log("Hello World"); // Printing a number console.log(10); // Printing a calculation console.log(5 + 3);
Console Output
Hello World
10
8

04 — Variables

Storing data
with variables

A variable is like a labeled box — you put a value inside it, give it a name, and later you can use that name to get the value back.

Keyword

let

Tells JavaScript: "I want to create a box to store something."

Name

The label

You choose it. Use clear, descriptive names like name or age.

script.js
let name = "Bilal"; let age = 20; console.log(name); console.log(age);
Console Output
Bilal
20
Notice

Text vs Numbers

Text goes inside quotes "Bilal". Numbers don't need quotes: 20.

05 — Variables

Variables
can change.

After you create a variable with let, you can give it a new value at any time — just write the name and assign a new value. No need to use let again.

Key rule

Use let only once per variable

When you first create it: let score = 10;
To update later: score = 15; — no let.

script.js
let score = 10; // Later... the score changes: score = 15; console.log(score);
Console Output
15

The value 10 is gone — the variable now holds 15.

06 — User Input

Asking the user
with prompt()

prompt() opens a small dialog box that asks the user a question. Whatever they type is returned — you store it in a variable.

Step 1

Ask the question

prompt("Your question")

Step 2

Store the answer

let x = prompt(...)

script.js
let name = prompt("What is your name?"); console.log(name);
script.js
let age = prompt("How old are you?"); console.log(age);
If user types "Bilal" then "20"
Bilal
20

07 — Math

Doing calculations
with operators

+
Addition
−
Subtraction
*
Multiply
/
Divide
%
Remainder
script.js
let a = 10; let b = 3; console.log(a + b); console.log(a - b); console.log(a * b); console.log(a / b); console.log(a % b);
Console Output
13 ← 10 + 3
7 ← 10 − 3
30 ← 10 × 3
3.333... ← 10 ÷ 3
1 ← remainder of 10 ÷ 3
Tip — %

The modulo operator

10 % 3 = 1 because 10 divided by 3 leaves a remainder of 1. Useful to check if a number is even or odd.
