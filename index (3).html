Concept 01

console.log()

console.log() prints a value to the developer console — your window into what the code is doing.

script.js
console.log("Hello World"); console.log(5); console.log("I am learning JavaScript");
Console output
Hello World
5
I am learning JavaScript
Text — strings
Text always goes inside quotes.
"Hello" or 'Hello'
Numbers
Numbers go in directly — no quotes.
5  3.14  100
Concept 02

Variables with let

A variable is a named container for a value. Name it once, use it everywhere.

script.js
let name = "Bilal"; let age = 20; console.log(name); // Bilal console.log(age); // 20
Keyword — let
Creates a new variable. Write it only once — when you first declare it.
Naming — camelCase
firstName  totalScore  itemPrice
Updating a variable
After declaring, update without let:  score = 15;

let a = 10; let b = 3; console.log(a + b); // 13 console.log(a % b); // 1 ← remainder of 10 ÷ 3
Concept 04

Reading input with prompt()

prompt() shows a popup asking the user to type something. Whatever they type gets stored in a variable your program can use.

script.js
let name = prompt("What is your name?"); console.log(name);
What happens
What is your name?  ← popup appears, user types: Bilal
Bilal  ← stored and printed
The 3 steps
1. Popup appears
2. User types a value
3. Value stored in variable
⚠ For numbers
prompt() always returns text. Wrap it in Number() to use it for math:

let age = Number(prompt("Age?"));
Concept 05  ·  Full program

Your first complete program

Combine all four concepts: input → store → calculate → output.

  • step 1Ask the user for input with prompt()
  • step 2Convert text to number with Number()
  • step 3Calculate and store the result in a variable
  • step 4Print the result with console.log()
salary-calculator.js
// Step 1 & 2 — get input let salary = Number(prompt("Enter your salary:")); let bonus = Number(prompt("Enter your bonus:")); // Step 3 — calculate let total = salary + bonus; // Step 4 — output console.log("Final salary:"); console.log(total);
Example run — user enters 3000 and 500
Enter your salary: 3000
Enter your bonus: 500
Final salary:
3500

Code writing Challenge 09  ·  Write the full program

Write the program

📋 Task
A teacher distributes books to students. Write a program that:
  1. Asks the user for the number of books (use Number(prompt(...)))
  2. Asks the user for the number of students
  3. Calculates each — how many books per student (books / students)
  4. Calculates leftover — remaining books (books % students)
  5. Prints each then leftover

Test values: books = 29, students = 6 → should print 4.833… then 5
your-solution.js
TAB = 2 spaces
unlock answers first — click the lock in the top bar
solution.js
let books = Number(prompt("Number of books?")); let students = Number(prompt("Number of students?")); let each = books / students; let leftover = books % students; console.log(each); // 4.833… (29 ÷ 6) console.log(leftover); // 5 (29 % 6)
