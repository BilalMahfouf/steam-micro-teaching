# stem-micro-teaching

Interactive micro-teaching slides for a beginner JavaScript session focused on input, output, variables, and basic math.

## Repository contents

- `index (7).html`: A self-contained presentation app (HTML, CSS, and JavaScript in one file).

## What this project includes

- 13 slide flow with keyboard/button navigation
- Progress indicator and session top bar
- Instructor PIN lock/unlock for revealing solutions
- Interactive student activities:
  - Output prediction
  - Code line ordering
  - Code writing and output checking
- Light/dark theme toggle

## How to use

1. Open `index (7).html` in a web browser.
2. Navigate slides with:
   - On-screen **prev/next** buttons
   - Keyboard **Left/Right arrows** or **Space**
3. Use the lock control to open the instructor PIN modal and unlock answers when needed.

## Notes

- This is a static project with no external build or test pipeline in the repository.
- Update the PIN in the script section (`CORRECT_PIN`) before classroom use.
